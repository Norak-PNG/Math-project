Classical Cryptography

Caesar Cipher

The Caesar Cipher is one of the simplest and oldest methods of encrypting messages, named after Julius Caesar, who reportedly used it to protect his military communications. It works by shifting each letter in the plaintext by a fixed number of positions down or up the alphabet.

How It Works:
1. Letter to Number Conversion: Each letter is first converted to a number (A=0, B=1, ..., Z=25).
2. Encryption: The encryption function is f(p) = (p + key) mod 26, where:

  • p is the numerical value of the plaintext letter,
  • key is the number of positions to shift (e.g., 3),
  • mod 26 ensures the result wraps around the alphabet.
For example, if you shift 'A' (0) by 3, you get 'D' (3). So, "HELLO" becomes "KHOOR" when the key is 3.

3. Decryption: To reverse the encryption, use the function f(c) = (c - key + 26) mod 26, where:

  • c is the numerical value of the ciphertext letter,
  • Adding 26 ensures the result stays positive before applying mod 26.
For example, if you decrypt 'D' (3) with a key of 3, you get 'A' (0).

Caesar Cipher is useful for basic demonstrations of encryption but is not secure by modern standards due to its predictable nature and small key space.

How it works:

  • Key Generation: Choose two prime numbers, multiply them to get n. Then calculate e (public exponent) and d (private exponent).
  • Encryption: Convert the message to a number m. Then compute c = (me) mod n.
  • Decryption: Compute m = (cd) mod n using the private key.
